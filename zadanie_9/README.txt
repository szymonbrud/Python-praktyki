Jak działa:
  - w folderze assets jest folder clientDisc, wszysko co wrzucimy do folderu zostanie przegrane do folderu disc w folderze server.
  - jeśli jakiś plik zostanie edytowany również jego aktualna wesja zostanie przesłana

Aby uruchomić program należy:
  - zainstalować potrzebne biblioteki
  - w folderze client uruchomić: python main.py
  - w folderze server uruchomić: python main.py
